     Inputs

    Metro UI CSS provides tools for styling input fields. Such as: text, password, checkbox, switch, etc. The input elements are formed from the outer container, the input element and auxiliary elements. This allows you to style the input elements, and give them additional futures.

    Text & password

    1. <!-- input[type=text] -->
    2. <div class="input-control text">
    3.     <input type="text">
    4. </div>
    6. <!-- input[type=password] -->
    7. <div class="input-control password">
    8.     <input type="password">
    9. </div>
    11. <!-- input with placeholder -->
    12. <div class="input-control text">
    13.     <input type="text" placeholder="Input you text here...">
    14. </div>
    16. <!-- readonly input -->
    17. <div class="input-control text">
    18.     <input type="text" readonly>
    19. </div>
    21. <!-- disabled input -->
    22. <div class="input-control text">
    23.     <input type="text" disabled>
    24. </div>
